### `.border-{value}`  
- **Description:** Sets the border-width property to the specified value for all sides of the element.  
- **CSS equivalent:**  
  ```css
  .border-{value} {
    border-width: {value}px;
  }
  ```  
- **Example usage:**  
  ```html
  <div class="border border-2">
    <!-- Content with border width of 2px -->
  </div>
  ```

  ---
