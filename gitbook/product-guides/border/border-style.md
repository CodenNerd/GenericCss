### `.border-dashed`  
- **Description:** Creates a dashed line border.  
- **CSS equivalent:**  
  ```css
  .border-dashed {
    border-style: dashed;
  }
  ```  
- **Example usage:**  
  ```html
  <div class="border-dashed">
    <!-- Element with a dashed border style -->
  </div>
  ```
