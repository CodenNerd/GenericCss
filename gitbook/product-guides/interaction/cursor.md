#### **`.cursor-auto`**

- **Description:** Sets the cursor to be automatically determined based on the element's context.
- **CSS equivalent:**
    ```css
    .cursor-auto {
      cursor: auto;
    }
    ```
- **Example usage:**
    ```html
    <div class="cursor-auto">Hover over me to see the auto cursor.</div>
    ```
