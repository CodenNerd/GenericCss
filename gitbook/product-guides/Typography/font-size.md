# üîç **Font Size Classes**

#### **`.text-{number}`**  
- **Description:** Sets the font size of the text.  
- **CSS equivalent:**  
  ```css
  font-size: {number}px;
  ```  
- **Example usage:**  
  ```html
  <span class="text-16">
    This text has a font size of 16 pixels.
  </span>
  ```  

    ---

#### **`.text-largest`**  
- **Description:** Sets the font size to the largest predefined size.  
- **CSS equivalent:**  
  ```css
  font-size: {largest};
  ```  
- **Example usage:**  
  ```html

  </span>
  ```  
