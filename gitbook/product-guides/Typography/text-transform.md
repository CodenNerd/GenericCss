#### **`.sentence-case`**  
- **Description:** Transforms the text to sentence case.  

- **CSS equivalent:**  
  ```css
  .text-sentence-case {
    text-transform: lowercase;
  }

  .text-sentence-case:first-letter {
    text-transform: uppercase;
  }
  ```  
- **Example usage:**  
  ```html
  <span class="sentence-case">This text is transformed to sentence case.</span>
  ```
