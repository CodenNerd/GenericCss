# ✍️ Font Style Classes

#### **`.font-italic`**  
- **Description:** Sets the font style to italic.  
- **CSS equivalent:**  
  ```css
  font-style: italic;
  ```  
- **Example usage:**  
  ```html
  <span class="font-italic">
    This text is italic.
  </span>
  ```  

    ---

#### **`.font-normal`**  
- **Description:** Sets the font style to normal.
