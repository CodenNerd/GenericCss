# 🌫️ Text Shadow Classes

#### **`.text-shadow-{color}`**  
- **Description:** Adds a text shadow with the specified color.  

- **CSS equivalent:**  
  ```css
  text-shadow: {value};
  ```  
- **Example usage:**  
  ```html
  <span class="text-shadow-red">This text has a red text shadow.</span>
  ```
